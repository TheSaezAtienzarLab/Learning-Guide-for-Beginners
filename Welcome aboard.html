How to start

There are different programming languages that will be used. For the applications in the Lab, we will focus on learning R, Phyton, and Linux Shell Scripting.

This guide aims to provide learning resources for R and R Studio.

Start with R for beginners

R is a programming language and a software environment for statistical computing and graphics.

1. Download and Install R and R studio

Download and install R and R Studio for your type of computer. This is some info on how to download it.

2. Install Swirl and start learning

Write the following command in the R Studio command and press 1. This will take you to R Programming: The basics of programming in R.

install.packages("swirl")
library("swirl")
swirl()
1

Follow the prompts to continue learning.
More information about Swirl can be found here.
